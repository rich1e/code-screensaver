# Matrix Overlay

A Matrix-style animated overlay for the VS Code editor area.

## Features

- Full editor-area overlay
- Classic Matrix "digital rain" animation
- Toggle via command
- ESC to exit instantly

## Usage

1. Open Command Palette
2. Run `Matrix Overlay: Start`
3. Press `ESC` to exit

## Notes

- This is a visual / aesthetic plugin
- It does not modify your code
- Animation pauses when closed
