# Code Screensaver

When you leave your desk, it shows a cool Matrix-style animation to hide your code from prying eyes.

## Features

- Full editor-area overlay
- Classic Matrix "digital rain" animation
- Toggle via command
- ESC to exit instantly

## Usage

1. Open Command Palette
2. Run `Matrix Overlay: Start`
3. Press `ESC` to exit

## Config

```json
"matrixOverlay.opacity": 0.1, // opacity
"matrixOverlay.fps": 20,      // fps
"matrixOverlay.fontSize": 24  // font size
```

## Notes

- This is a visual / aesthetic plugin
- It does not modify your code
- Animation pauses when closed
